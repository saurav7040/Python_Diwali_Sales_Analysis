# 🪔 Python Diwali Sales Analysis

## 📌 Project Overview
This project analyzes Diwali sales data using Python to uncover customer purchasing behavior and business insights. The analysis helps identify target customers, popular product categories, and sales trends that can support data-driven business decisions.

## 🎯 Objectives
- Clean and preprocess sales data
- Perform Exploratory Data Analysis (EDA)
- Identify key customer segments
- Analyze sales trends across demographics
- Visualize insights using charts and graphs

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset Features
The dataset contains:
- User ID
- Gender
- Age Group
- Marital Status
- State
- Occupation
- Product Category
- Orders
- Amount

## 📊 Analysis Performed
### Data Cleaning
- Removed null values
- Dropped unnecessary columns
- Checked data types and corrected inconsistencies

### Exploratory Data Analysis
- Gender-wise sales analysis
- Age group analysis
- State-wise sales performance
- Marital status analysis
- Occupation-based analysis
- Product category analysis

## 📈 Key Insights
- Female customers contributed the highest sales.
- Customers aged 26–35 years were the most active buyers.
- Married women formed the largest customer segment.
- Maharashtra, Uttar Pradesh, and Karnataka generated the highest sales.
- Food, Clothing, and Electronics were the top-selling product categories.

## 🚀 Business Impact
The insights from this analysis can help businesses:
- Improve marketing strategies
- Target high-value customer groups
- Optimize inventory management
- Increase customer satisfaction and sales revenue
